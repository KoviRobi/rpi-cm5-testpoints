# Raspberry Pi CM5 testpoints

The [Raspberry Pi CM5
datasheet](https://www.raspberrypi.com/documentation/computers/compute-module.html#specifications)
contains the test point map in appendix A.1, but it's hard to
visualise. So I have created this following page to help visualise it.

View it at <https://kovirobi.github.io/rpi-cm5-testpoints/>
